# 📘 演算法基本概念（Big-O、時間複雜度、APCS 必備分析）

演算法（Algorithm）是一組明確、可重複的步驟，用來解決問題。

## 🧭 1. 什麼是 Big-O 記號？

Big-O 描述「程式在資料量 n 變大時，執行時間變慢的速度」。

---

## 🧪 2. 常見時間複雜度（由快到慢）

| Big-O | 名稱 | 範例 | 說明 |
|-------|------|------|-------|
| O(1) | 常數 | 取 a[3] | 不變 |
| O(log n) | 對數 | 二元搜尋 | 每次砍一半 |
| O(n) | 線性 | for 迴圈 | 資料越多越慢 |
| O(n log n) | n × log n | 快速排序 | APCS 必考 |
| O(n²) | 平方 | 雙層迴圈 | 易 TLE |
| O(2ⁿ) | 指數 | 子集合枚舉 | n=30 爆 |
| O(n!) | 階乘 | 全排列 | n=12 爆 |

---

## 🔍 3. 如何估算時間複雜度？

### ✔ 3-1：看迴圈層數

```cpp
for(int i=0;i<n;i++)
    for(int j=0;j<n;j++)
```

→ **O(n²)**

### ✔ 3-2：看資料是否每次「砍一半」

```python
while n > 1:
    n //= 2
```

→ **O(log n)**

### ✔ 3-3：只留最大項

```
O(n) + O(log n) = O(n)
```

## 🧠 4. 常見範例

### 🌟 例 1：O(n) vs O(n²)

```cpp
O(n)
O(n*n)
```

→ O(n) 較快

---

### 🌟 例 2：加法 vs 乘法迴圈

```cpp
for(int i=0;i<n;i++)     // O(n)
for(int i=1;i<n;i*=2)    // O(log n)
```

---

### 🌟 例 3：複合迴圈

```cpp
for(int i=0;i<n;i++)
    for(int j=0;j<i;j++)
```

→ 平均 n/2 次 → **O(n²)**

## 📚 7. 重點觀念

- 比成長速度，不比秒數  
- Big-O 只留最大項  
- APCS 最常考：O(n)、O(n log n)、O(n²)  
- 看程式推複雜度是核心能力
