# 演算法課程總覽（Algorithms Overview）

本單元整理資訊課程中最重要的演算法主題，提供清楚易讀的教學內容，  
適合同時修習 C++ / Python 的同學使用。

---

## 📘 本單元內容架構

- **基本概念（Big-O）**  
  認識演算法效率、時間複雜度與空間複雜度。

- **搜尋與圖論（BFS / DFS）**  
  包含走迷宮、最短路徑、找連通元件、拓樸排序等常見題型。

- **排序與資料結構**  
  各種排序方法、Stack / Queue / 優先佇列、樹與二元搜尋樹等。

---

## 📌 建議學習順序

1. **Big-O + 基礎時間複雜度**
2. **BFS / DFS + 圖論題型**
3. **排序與基本資料結構**
4. 與 APCS / TQC+ / ZeroJudge 題庫練習結合

祝同學在演算法的世界越走越順利！
