# C++ 教學與資源

本頁整理 C++ 程式語言的基礎教學、線上工具、語法參考與練習平台。  
適用對象：資訊課 C++、APCS、NPSC、ZeroJudge 初學者與進階者。

---

## 📘 1. C++ 入門教學（初學者）
- **C++ Tutorial — W3Schools**  
  https://www.w3schools.com/cpp/  
  適合初學者的圖文式教學，清楚易懂。

- **C++ 教學筆記（NTU）**  
  https://www.csie.ntu.edu.tw/~cyy/courses/oop/  
  台大課程筆記，內容扎實。

- **cplusplus 語法教學（繁體）**  
  https://openhome.cc/Gossip/CppGossip/  
  非常適合高中學生閱讀，講解直白。
  
- **Yui Huang 演算法學習筆記**  
 https://yuihuang.com/
  非常適合高中學生閱讀，講解直白。
  

---

## 📙 2. C++ 語法查詢（Reference）
- **cppreference 官方語法查詢**  
  https://en.cppreference.com  
  最完整 C++ 語法與 STL 參考。

- **cplusplus.com 語法手冊**  
  https://cplusplus.com/reference/  
  查 STL、function、library 的好工具。

---

## 📗 3. C++ 競賽 / 練習平台
- **ZeroJudge**  
  https://zerojudge.tw  
  台灣最常用線上評測平台。

- **洛谷（Luogu）**  
  https://www.luogu.com.cn  
  題目多、分類完整，適合練習 DP、圖論等。

- **UVa Online Judge**  
  https://onlinejudge.org  
  老牌國際競賽題庫。

---

## 🧰 4. C++ 實作工具
- **OnlineGDB（可線上編譯 C++）**  
  https://www.onlinegdb.com

- **Compiler Explorer**  
  https://godbolt.org  
  可查看程式編譯後的組語，非常適合學習優化。
